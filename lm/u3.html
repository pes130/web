Fragmento HTML 0

Introducción al modelo de cajas de CSS

Estilo


	/* Borde para el html */
	html{border: 3px solid green;}
	/* borde para el body */
	body {border: 3px dotted; text-align: center;}
	/* borde para p, h1 y h2 (elementos de bloque) */
	p,h1,h2 {border: 1px dotted;	text-align: center; margin:10px; padding:10px}
	/* borde para strong, em y spam (elementos de línea) */
	strong, em, span {border: 1px dashed red;margin: 5px;}

HTML

	<h2>Caja 2</h2>

	</p>
